# Скрипты для проекта «8 марта»

## Обновление списка коллег из Excel

1. Положите файл **Поздравление.xlsx** в папку `~/Downloads` (или укажите путь в скрипте).
2. В xlsx должны быть колонки: **id**, **Почта**, **Имя**, **Текст поздравления** (лист 2).
3. Запустите:
   ```bash
   python3 scripts/xlsx_to_colleagues.py
   ```
4. Скопируйте вывод в `index.html`, заменив массив `COLLEAGUES_TABLE`.

Фото подставляются по id: для строки с id=5 используется файл `photos/5.png`. Добавьте в папку `photos/` файлы `1.png`, `2.png`, … по числу id в таблице.
